CSS Properties Field

Input Type: Text

Format: Values separated by semicolons (;)

What it does: Applies multiple CSS properties to the box in sequence.

Property Order:

  1. Background color/image
  2. Border radius
  3. Border color
  4. Border width
  5. Height
  6. Width

Example: #ff5733;20px;blue;4px;350px;350px

Gradient Field

Input Type: Text

Format: Color values separated by semicolons (;)

What it does: Creates a linear gradient background with the specified colors. Supports up to 8 colors.

Example: red;orange;yellow;green;blue

Note: If only one color is provided, it will be applied as a solid background.

Color Picker

Input Type: Color selector

What it does: Allows you to visually select a color and apply it as the box background.

Features:

  • Visual color picker interface
  • Real-time hex code display
  • Automatic text color contrast adjustment
  • Click "Set Color" button to apply

Reset Button

What it does: Resets all properties and inputs to their default values, clearing the box customization.

Theme Toggle

What it does: Switches between light and dark themes. Theme preference is saved in browser local storage.

Size Limits

Current limits: 120px to 640px (auto-adjusts to your screen).

Applied to: Height and width values set through CSS Properties.
